How to Run the Application :-

This web application uses FLASK 1.1.2 web framework.

To run the application, flask should be installed.

Use the following commands in the terminal opened in the application directory to run the application:

set FLASK_APP=flashcards.py
set FLASK_ENV=development
flask run

The first 2 commands are to be done the first time you run the application. Once set, the 3rd command is to run the application.

Note: These commands were used to run the application on a windows computer and may not work on other OSs.
